# Decora User Stories

## Customer Stories

### US-001 - User Registration

As a customer,

I want to create a new account

So that I can place orders and track my purchases.

---

### US-002 - User Login

As a customer,

I want to sign in

So that I can access my account securely.

---

### US-003 - Browse Products

As a customer,

I want to browse products by category

So that I can easily find the products I need.

---

### US-004 - Search Products

As a customer,

I want to search for products

So that I can quickly find a specific item.

---

### US-005 - View Product Details

As a customer,

I want to view product details

So that I can make an informed purchasing decision.

---

### US-006 - Add to Cart

As a customer,

I want to add products to my shopping cart

So that I can purchase multiple products together.

---

### US-007 - Checkout

As a customer,

I want to complete the checkout process

So that I can place my order successfully.

---

### US-008 - Track Orders

As a customer,

I want to track my orders

So that I know the current delivery status.

## Admin Stories

### US-101 - Manage Products

As an administrator,

I want to create, update, and delete products

So that customers always see accurate product information.

---

### US-102 - Manage Categories

As an administrator,

I want to manage product categories

So that products remain organized.

---

### US-103 - Manage Orders

As an administrator,

I want to update order statuses

So that customers receive accurate order information.

---

### US-104 - Manage Inventory

As an administrator,

I want to update product stock

So that unavailable products cannot be ordered.
